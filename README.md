# JRuby + Swing + Grafico #
***
### Pequena Descricao ###
Apos incansáveis tentativas de usar outras GUIs do ruby, percebi q se eu quisesse
que funcionasse direito em muitas plataformas, só conseguiria com JRuby + Swing.
Como sou totalmente leigo em Swing, peguei um exemplo feito em java, e passei esse
exemplo para JRuby.xD
Eu, Pessoalmente, acho bizarra a maneira com que o Swing trabalha com grafico,
porem achei que usa-lo seria o melhor.

O código java original pode ser encontrado em...
http://www.phy.ohiou.edu/~mousseau/phy303/scientific/all_33.html
